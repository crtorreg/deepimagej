DeepImageJ plugin

  • Download

The ZIP file is a plugin a ImageJ or Fiji. It contains all the necessary libraries (JAR files) to load and run TensorFlow models on any OS: Windows, Mac OSX, Linux.

One-click Installation: Unzip the ZIP file and store the 5 JAR files into the plugins folder of ImageJ or Fiji.

How to use: Check the Documentation of the plugin.

Source code: Github Java source code of the plugin.
