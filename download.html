DeepImageJ plugin

  • Download

The ZIP file is a plugin for ImageJ or Fiji. It contains all the necessary libraries (JAR files) to load and run TensorFlow models on any OS: Windows, Mac OSX, Linux.

One-click Installation: Unzip the ZIP file and store the 5 JAR files into the plugins folder of ImageJ or Fiji.

How to use: Check the Documentation of the plugin.

Source code: GitHub Java source code of the plugin.

The ZIP file consists of 3 plugins: Windows, Mac OSX, Linux.

DeepImageJ Run: This plugin applies the neural network to an input image (it is macro-recordable).

DeepImageJ Explore: This plugin allows to explore all the installed models.

DeepImageJ BuildBundledModel: This plugin allows to build a bundled models from a TensorFlow models, no coding is needed.

Converter Keras to TensorFlow
