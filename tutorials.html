General description

DeepImageJ allows any user without previous programming knowledge to apply deep learning models to their image processing tasks. Moreover, it is a user-friendly tool that contributes to the spread of deep learning models, breaking the existing technological barriers between computer science and applied science.

The software stores the conditions and parameters needed to make inference on any given image in a file along with the actual model. This file is retrieved every time the model is called, allowing the user to apply it with no knowledge about the characteristics of the neural network architecture.
