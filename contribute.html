Contributions

Anyone can contribute by building a bundled model using DeepImageJ Build BundledModel and uploading it to the online repository.

Before advertising and providing any bundled model, all contributions are tested by DeepImageJ developers. Therefore, if you want to contribute, please, write an email to DeepImageJ with the following information:

  • Name and Surname of the contributor
  • Contact e-mail address
  • A short description of the model and the targeted task
  • A ZIP file containing the bundled model (Tensorflow SavedModel, pre- and post-processing macros and config.xml file)
  • Input and output example images. These can be provided within the ZIP file
